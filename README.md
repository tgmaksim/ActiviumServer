# Gymnasium App

Официальный репозиторий приложения для **Гимназии №147 (г. Омск)**.  
Проект объединяет школьные сервисы в одном приложении, а также являются продуктом итогового проекта.
---

## Сайт проекта

Сайт используется для:
- скачивания Android-приложения
- ознакомления с функциональностью
- отображения скриншотов
- информирования об обновлениях

Сайт адаптирован под мобильные устройства и широкие экраны, поддерживает светлую и тёмную темы.
