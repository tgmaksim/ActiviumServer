# Activium API (+ сайт)

API и сайт для школьного приложения «Активиум»

## Технологии
- Python 3.12
- FastAPI
- SQLAlchemy 2.x (async)
- PostgreSQL (asyncpg)

## Сайт
- HTML-страницы находятся в templates
- Статические файлы находятся в папке www и должны передаваться низкоуровневым сервером (uwsgi)

## API
- Документация находится по пути /<API_PREFIX>/docs и хранит в себе описание каждого метода
- Любой API-метод требует api_key в заголовке http-запроса
- Ответом сервера всегда является JSON из сериализованных сущностей (каждая сущность имеет поле classId - идентификатор класса сущности)
